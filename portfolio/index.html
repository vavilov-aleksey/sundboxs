Обо мне

 

Доброго времени суток, меня зовут Алексей. Занимаюсь WEB разработкой сайтов 1.5 года, в основном Frontend.
Хорошие знания html,css,js. php на уровне написания небольшого скрипта. В качестве среды разработки использую PhpStorm. Знаком и использую на практике Git, с ветвлением Git-Flow.
Основную часть своего рабочего времени верстаю сайты с Psd макетов. Хорошо знаком с Photoshop, Figma, Zeplin.
Вёрстку делаю адаптивную, кроссбраузерную. В качестве шаблонизатора html использую jade(pug). Для написания Css использую препроцессор Sass(Scss). Собираю проект с помощью сборщика Gulp.
Постоянно изучаю новые технологии, открываю для себя интересные библиотеки и фреймворки. Заинтересован в развитии и изучении Frontend направления. В настоящее время изучаю документацию по фреймворку React.


P.S В данный момент работаю frontend разработчиком в it-компании. Компания занимается разработкой новых проектов, и поддержкой существующих.
